Welcome to Theanet!
===================

Theanet is a wrapper around Theano. It is a Theano based Convolutional Neural
Network library for image classification with

- Elastic Distortion

- Noising of inputs

- Dropout

- L1, L2, Maxnorm Regularization

- Softmax, Mixture of Gaussians outputs

- etc.

The project needs more testing and documentation. The project is at `GitHub`_.
Contributions are welcome!
